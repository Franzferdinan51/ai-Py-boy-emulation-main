# OpenClaw Agent - GameBoy Emulator

Control Game Boy, Game Boy Color, and Game Boy Advance games via MCP.

## Tools

- `load_rom` - Load ROM
- `press_button` - Press button
- `get_screen` - Get screen image
- `get_state` - Get emulator state
- Memory reading available

## Setup

```bash
mcporter add gameboy --stdio "python3 mcp_server.py"
```
